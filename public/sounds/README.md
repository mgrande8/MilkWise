# Sound Files

Download royalty-free audio files from Pixabay.com or Freesound.org and place them here:

- rain.mp3 - Gentle rain sounds (2-3 min loop)
- ocean.mp3 - Ocean waves (2-3 min loop)  
- forest.mp3 - Birds and forest ambiance (2-3 min loop)
- whitenoise.mp3 - Soft white noise (2-3 min loop)

Recommended sources:
- https://pixabay.com/sound-effects (free, no attribution)
- https://freesound.org (check licenses)
